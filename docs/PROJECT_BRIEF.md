# OpsPilot AI - Project Brief

## Goal

Build a production-style internal operations workflow platform with AI-assisted features.

This project is intended as a portfolio project that demonstrates real-world full stack engineering skills.

## Target Users

- Admin
- Operations Manager
- Staff / Reporter

## Core MVP

1. Authentication
2. Role-based access control
3. Incident / task creation
4. Status workflow
5. Comments
6. Attachments
7. Audit logs
8. Dashboard
9. AI summary and classification
10. Semantic search

## Initial Tech Stack

- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: FastAPI, SQLAlchemy async, Alembic
- Database: PostgreSQL
- Local Dev: Docker Compose
- Testing: pytest
- CI/CD: GitHub Actions later
